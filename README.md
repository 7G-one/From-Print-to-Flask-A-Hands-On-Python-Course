# 🐍 Python 板书教学课程

> 从 print("Hello World") 到搭建 Flask Web 应用

## 这是什么？

一套给**想学 Python 但不知道从哪开始**的人准备的课程。

不是那种"我给你看代码你自己悟"的课——每节课都有**可运行的实战项目、3 道练习题、生活类比**，17 课带你从 `print()` 走到能自己写一个 Flask Web 应用。

函数在第 2 课就教了（不是第 8 课），闭包和装饰器合并在一起讲（不是分开两课），每课都有完整的代码能直接跑。学到最后你会明白：为什么 Python 是最好的第一门编程语言。

## 怎么跑？

```bash
# 进入课程目录
cd "d:/VS project/Mimo project/s_python"

# 运行任意一课
python 00_零基础入门.py
python 01_变量与数据类型.py
...
python 17_Web开发入门.py
```

## 几个小建议

1. **别跳课** — 后面的课用到前面的知识，跳了会懵
2. **别光看** — 直接运行，改参数，看结果怎么变
3. **做练习** — 练习题不是摆设，先自己写再看答案
4. **用函数** — 第 2 课就教了函数，后面所有课都用函数组织代码
5. **别怕报错** — Python 的错误提示很友好，认真读就能看懂

## 环境要求

- **Python 3.7+** — 从 [python.org](https://python.org) 下载
- **编辑器** — 推荐 VS Code + Python 扩展

## 课程特色

- 🎯 **生活类比** — 每课都有，把抽象概念变成日常例子
- 📝 **逐行注释** — 每行代码都有详细说明
- 🧮 **实战项目** — 不是孤立代码片段，是能跑起来的完整程序
- ⚠️ **常见陷阱** — 标注容易犯的错误，避免踩坑
- 📋 **练习题** — 每课 3 道，有题目+提示+参考答案
- 🔧 **函数优先** — 第 2 课就教函数，后续所有课用函数组织代码
